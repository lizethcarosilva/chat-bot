
 ARCHIVOS GENERADOS AL ENTRENAR LA IA


chat-bot/

 models/   CARPETA DONDE SE GUARDAN LOS MODELOS ENTRENADOS
   
    ==================== MODELOS DE PREDICCIÓN ====================
   
    predictor_model.h5           (1-2 MB)
      > Red neuronal para predecir tipo de mascota y asistencia
          • Arquitectura: Dense  Dropout  Dense  Softmax
          • Pesos entrenados (miles de parámetros)
          • Configuración del optimizador
   
    scaler.pkl                   (10-50 KB)
      > Normalizador y codificadores
          • StandardScaler (normaliza features)
          • LabelEncoder (convierte tipos a números)
   
    ==================== MODELOS DEL CHATBOT ====================
   
    chatbot_veterinario.h5       (3-5 MB)
      > Red neuronal LSTM para clasificar intenciones
          • Bidirectional LSTM (64 unidades)
          • Embeddings (5000 palabras x 128 dimensiones)
          • Pesos entrenados
   
    tokenizer_veterinario.pkl    (100-200 KB)
      > Vocabulario del chatbot
          • Diccionario: palabra  número
          • Ejemplo: "perro"  1, "gato"  2
   
    label_encoder_veterinario.pkl (5-10 KB)
      > Codificador de intenciones
          • Array: ['saludo', 'sintomas', 'vacunas', ...]
          • Convierte intenciones a índices
   
    intents_veterinario.pkl      (20-100 KB)
       > Diccionario de respuestas predefinidas
           • {'saludo': ["Hola!", "Bienvenido!"], ...}

 data/     DATASETS (opcional)
    dataset_citas_ml.csv         (si exportas datos)

 exports/  EXPORTACIONES (opcional)
     analisis_clustering.json     (si guardas análisis)



 PROCESO DE GUARDAR Y CARGAR


ENTRENAR  GUARDAR  CARGAR  USAR
                              
  5-10 min   disco    inicio   API


1. ENTRENAR (Una sola vez)
   
   python entrenar_chatbot_veterinario.py
   
   Crea archivos en models/


2. GUARDAR (Automático)
   
   model.save('models/chatbot_veterinario.h5')
   
   Escribe modelo en disco


3. CARGAR (Al iniciar API)
   
   model = load_model('models/chatbot_veterinario.h5')
   
   Lee modelo desde disco a memoria


4. USAR (En cada predicción)
   
   prediction = model.predict(texto)
   
   Usa modelo en memoria (rápido)



 INFORMACIÓN DE VALIDACIÓN (Solo en consola)


Durante el entrenamiento verás:

Epoch 1/50
  loss: 2.1234         Error en datos de entrenamiento
  accuracy: 0.4562     Precisión en entrenamiento
  val_loss: 2.3456     Error en datos de validación
  val_accuracy: 0.4325  Precisión en validación  (LA MÁS IMPORTANTE)

...

Epoch 50/50
  loss: 0.2134
  accuracy: 0.9425     94.25% en entrenamiento
  val_loss: 0.2567
  val_accuracy: 0.9125  91.25% en validación 


INTERPRETACIÓN:


val_accuracy > 90%   Excelente 
val_accuracy > 80%   Bueno 
val_accuracy > 70%   Aceptable 
val_accuracy < 70%   Necesita mejora 

Si (accuracy - val_accuracy) > 10%  Sobreajuste (overfitting)
  Solución: Más datos, más dropout, regularización


ESTA INFORMACIÓN NO SE GUARDA AUTOMÁTICAMENTE
Solo aparece en la consola durante el entrenamiento
