
 CÓMO ENTRENAR LA IA - GUÍA ULTRA SIMPLE


HAY 2 FORMAS:



 FORMA 1: ENTRENAR PREDICTOR (La Más Fácil)


PASO 1: Inicia la API

python api.py


PASO 2: Abre Swagger UI

http://localhost:8000/docs


PASO 3: Ejecuta Entrenamiento

1. Busca: POST /api/entrenar
2. Click: "Try it out"
3. Click: "Execute"


PASO 4: Espera 5-10 minutos

Verás el progreso en la consola de la API


PASO 5: Verifica

http://localhost:8000/api/predicciones/estado

Debe decir: "modelos_entrenados": true


¡LISTO! 



 FORMA 2: ENTRENAR CHATBOT LSTM (Más Avanzado)


PASO 1: Instala TensorFlow

pip install tensorflow


PASO 2: Ejecuta Script

python entrenar_chatbot_veterinario.py


PASO 3: Espera 5-10 minutos

Verás el progreso en consola:

Epoch 1/50 - accuracy: 0.45
Epoch 2/50 - accuracy: 0.58
...
Epoch 50/50 - accuracy: 0.96 


PASO 4: Verifica Archivos

dir models

Deberías ver:
  chatbot_veterinario.h5 
  tokenizer_veterinario.pkl 
  label_encoder_veterinario.pkl 


PASO 5: Reinicia API

Ctrl+C
python api.py


¡LISTO!
