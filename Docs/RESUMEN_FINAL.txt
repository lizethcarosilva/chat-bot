 REDES NEURONALES DEL SISTEMA



 RED 1: CHATBOT VETERINARIO (LSTM)                                          

                                                                             
 PROPÓSITO: Clasificar intenciones en preguntas de texto                    
                                                                             
 ARQUITECTURA:                                                               
   Input: "mi perro tiene fiebre"                                           
                                                                            
   Tokenización: [45, 12, 3, 67, 0, 0, ...]                                
                                                                            
   Embedding(5000, 128)   Convierte palabras a vectores                    
                                                                            
   Bidirectional LSTM(64)  Procesa secuencia en ambas direcciones          
                                                                            
   Dropout(0.5)           Previene overfitting                             
                                                                            
   Bidirectional LSTM(32)  Segunda capa de memoria                         
                                                                            
   Dense(64, relu)        Aprende patrones                                 
                                                                            
   Dense(n_clases, softmax)  Probabilidades por intención                  
                                                                            
   Output: {"enfermedad_perros": 0.85, "vacunas": 0.10, ...}               
                                                                             
 DATOS: datos_veterinarios.json (15+ intenciones médicas)                   
 PRECISIÓN: 85-95%                                                           
 ARCHIVO: models/chatbot_veterinario.h5                                     
                                                                             



 RED 2: PREDICTOR DE TIPO DE MASCOTA (Dense)                                

                                                                             
 PROPÓSITO: Predecir tipo de mascota según día, hora, servicio              
                                                                             
 ARQUITECTURA:                                                               
   Input: [dia_semana=5, hora=10, mes=11, service_id=1]                    
                                                                            
   Normalización (StandardScaler)                                           
                                                                            
   Dense(128, relu)  Primera capa oculta                                   
                                                                            
   Dropout(0.3)                                                              
                                                                            
   Dense(64, relu)   Segunda capa oculta                                   
                                                                            
   Dense(32, relu)   Tercera capa oculta                                   
                                                                            
   Dense(n_clases, softmax)  Probabilidades por tipo                       
                                                                            
   Output: {"Perro": 0.78, "Gato": 0.15, "Ave": 0.05}                      
                                                                             
 DATOS: Base de datos PostgreSQL (tabla appointment + pet)                  
 PRECISIÓN: 75-85%                                                           
 ARCHIVO: models/predictor_model.h5

 ANÁLISIS DE BASE DE DATOS


El sistema analiza datos de PostgreSQL y proporciona:

 Tipo de mascota más común
   Endpoint: GET /api/analisis/tipos-mascota
   Ejemplo: {"tipo_mas_comun": "Perro", "porcentaje": 45.5}

 Día con más atención
   Endpoint: GET /api/analisis/dias-atencion
   Ejemplo: {"dia_mas_atencion": "Viernes", "total_citas": 120}

 Horas pico
   Endpoint: GET /api/analisis/horas-pico
   Ejemplo: {"hora_pico": 10, "total_citas": 120}

 Servicios más utilizados
   Endpoint: GET /api/analisis/servicios
   Ejemplo: {"servicio_mas_popular": "Consulta General"}



 PREDICCIONES CON IA


 Predecir tipo de mascota para un día/hora específico
   Endpoint: POST /api/predicciones/tipo-mascota
   Entrada: {dia_semana: 5, hora: 10, mes: 11, service_id: 1}
   Salida: {"tipo_mas_probable": "Perro", "confianza": 0.78}

 Predecir probabilidad de asistencia
   Endpoint: POST /api/predicciones/asistencia
   Entrada: {dia_semana: 3, hora: 14, mes: 11, service_id: 2, edad_mascota: 5}
   Salida: {"probabilidad_asistencia": 0.82, "asistira": true}



 EJEMPLOS DE CONVERSACIÓN CON EL CHATBOT


Usuario: "Mi perro tiene fiebre"
Bot: "La temperatura normal en perros es 38-39°C. Si supera 39.5°C, es fiebre.
      Síntomas: letargo, pérdida de apetito, nariz caliente. Si es > 40°C,
      acude al veterinario inmediatamente."

Usuario: "Qué vacunas necesita un cachorro"
Bot: "Calendario de vacunación:
      • 6-8 semanas: Primera dosis múltiple
      • 10-12 semanas: Segunda dosis
      • 14-16 semanas: Tercera dosis + Rabia
      • 1 año: Refuerzo anual"

Usuario: "Mi gato vomita"
Bot: "Vómito en gatos puede ser por: comer rápido, cambio de alimento,
      bolas de pelo, parásitos. Si es persistente (>24h), con sangre,
      o acompañado de letargo, consulta al veterinario urgente."

Usuario: "Cuál es el tipo de mascota más común"
Bot: " El tipo más común es: Perro (45.5%)
      Estadísticas:
      • Perro: 250 (45.5%)
      • Gato: 180 (32.7%)
      • Ave: 70 (12.7%)"



 CÓMO AGREGAR MÁS INFORMACIÓN AL CHATBOT


PASO 1: Editar datos_veterinarios.json


Agrega una nueva intención:

{
  "tag": "nueva_enfermedad",
  "patterns": [
    "mi perro tiene otitis",
    "infección de oído",
    "mi mascota se rasca las orejas"
  ],
  "responses": [
    "La otitis es una inflamación del oído. Síntomas: rascado frecuente de 
     orejas, mal olor, secreción. Tratamiento: limpieza y gotas óticas 
     prescritas por veterinario. No uses remedios caseros."
  ]
}


PASO 2: Volver a entrenar

python entrenar_chatbot_veterinario.py


PASO 3: Reiniciar API

Ctrl+C (detener API)
python api.py (iniciar de nuevo)


¡Listo! El chatbot ahora responderá sobre otitis.
