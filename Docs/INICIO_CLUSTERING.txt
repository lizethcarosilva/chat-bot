 QUÉ HACE EL CLUSTERING


1. CLUSTERS DE MASCOTAS
   
   Identifica grupos de mascotas con características similares:
   
   Ejemplo:
   • Cluster 0: Perros jóvenes (< 2 años) con servicios preventivos
   • Cluster 1: Gatos adultos (3-7 años) con consultas rutinarias
   • Cluster 2: Mascotas senior (> 7 años) con tratamientos especializados


2. SEGMENTACIÓN DE CLIENTES
   
   Clasifica clientes en 4 segmentos:
   
   • VIP: Visitan frecuentemente, gastan mucho
       Estrategia: Programa de lealtad
   
   • Regular: Visitas moderadas, clientes fieles
       Estrategia: Mantener satisfacción
   
   • Ocasional: Visitas raras
       Estrategia: Campañas de reactivación
   
   • Nuevo: Recién llegados
       Estrategia: Onboarding y bienvenida


3. AGRUPACIÓN DE SERVICIOS
   
   Agrupa servicios con patrones similares:
   
   Ejemplo:
   • Grupo 0: Servicios matutinos de alta demanda
   • Grupo 1: Servicios vespertinos regulares
   • Grupo 2: Servicios especializados de baja frecuencia



 EJEMPLO EN REACT


import React, { useEffect, useState } from 'react';

function ClusteringDashboard() {
  const [data, setData] = useState(null);

  useEffect(() => {
    fetch('http://localhost:8000/api/clustering/completo')
      .then(res => res.json())
      .then(result => setData(result));
  }, []);

  if (!data) return <div>Analizando con IA...</div>;

  return (
    <div>
      <h1> Análisis de Clustering</h1>
      
      <h2> Segmentos de Clientes</h2>
      {data.clustering_clientes.segmentos.map(seg => (
        <div key={seg.segmento_id}>
          <h3>{seg.nombre}</h3>
          <p>{seg.total_clientes} clientes</p>
          <p>${seg.gasto_promedio.toFixed(2)} gasto promedio</p>
        </div>
      ))}
    </div>
  );
}

 INTERPRETACIÓN DE RESULTADOS


SILHOUETTE SCORE (Calidad):

  0.7 - 1.0   EXCELENTE
  0.5 - 0.7   BUENO
  0.3 - 0.5    MODERADO
  0.0 - 0.3   BAJO

Si el score es > 0.5  El clustering es útil y confiable
