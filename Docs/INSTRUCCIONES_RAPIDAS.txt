 INICIO RÁPIDO (3 PASOS)


PASO 1: Instalar dependencias

pip install -r requirements.txt

⏱ Tarda: 5-10 minutos


PASO 2: Entrenar chatbot veterinario (OBLIGATORIO)

python entrenar_chatbot_veterinario.py

⏱ Tarda: 2-5 minutos
 Genera: models/chatbot_veterinario.h5


PASO 3: Iniciar API

python api.py

 Servidor en: http://localhost:8000
 Docs: http://localhost:8000/docs

 CHATBOT VETERINARIO


El chatbot responde sobre:

 ENFERMEDADES:
   • Parvovirus, Moquillo, Rabia
   • Leucemia Felina, Inmunodeficiencia
   • Síntomas y tratamientos

 VACUNAS:
   • Calendarios de vacunación
   • Vacunas obligatorias
   • Dosis de refuerzo

 CUIDADOS:
   • Alimentación perros y gatos
   • Desparasitación
   • Higiene dental
   • Cuidados cachorros/gatitos

 EMERGENCIAS:
   • Síntomas de alerta
   • Primeros auxilios
   • Tóxicos para mascotas

EJEMPLOS DE PREGUNTAS:
   "Mi perro tiene fiebre"
   "Qué vacunas necesita un cachorro"
   "Mi gato vomita"
   "Cómo desparasitar a mi mascota"
   "Alimentos tóxicos para perros"

 CÓMO AGREGAR MÁS INFORMACIÓN AL CHATBOT


1. Edita: datos_veterinarios.json

2. Agrega nueva intención:

{
  "tag": "nombre_intencion",
  "patterns": [
    "pregunta ejemplo 1",
    "pregunta ejemplo 2",
    "pregunta ejemplo 3"
  ],
  "responses": [
    "Respuesta del chatbot aquí"
  ]
}

3. Vuelve a entrenar:
   python entrenar_chatbot_veterinario.py

4. Reinicia la API:
   Ctrl+C (detener)
   python api.py (iniciar)


EJEMPLO REAL:

{
  "tag": "otitis",
  "patterns": [
    "mi perro tiene otitis",
    "infección de oído",
    "mi mascota se rasca la oreja"
  ],
  "responses": [
    "La otitis es una inflamación del oído. Síntomas: rascado de orejas, mal olor, secreción. Tratamiento: limpieza y gotas óticas prescritas por veterinario."
  ]
}



 PROBAR EL CHATBOT


Opción 1: Desde la documentación interactiva

1. Abre: http://localhost:8000/docs
2. Busca: POST /api/chat
3. Click en "Try it out"
4. Escribe tu mensaje
5. Click "Execute"


Opción 2: Desde terminal (curl)

curl -X POST "http://localhost:8000/api/chat" \
  -H "Content-Type: application/json" \
  -d '{"mensaje": "mi perro tiene fiebre"}'


Opción 3: Desde tu frontend React

fetch('http://localhost:8000/api/chat', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ mensaje: "mi perro tiene fiebre" })
})
  .then(r => r.json())
  .then(data => console.log(data.respuesta));

 SOLUCIÓN DE PROBLEMAS


Error: "No se puede conectar a la base de datos"

• Verifica conexión a Internet
• Revisa .env tiene las credenciales correctas


Error: "Modelo de chatbot no encontrado"

python entrenar_chatbot_veterinario.py


Error: "ModuleNotFoundError: fastapi"

pip install fastapi uvicorn pydantic


Error de CORS desde React

En api.py línea ~38, ajusta:
allow_origins=["http://localhost:3000"]
